Important FAQ

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page as a tree of objects, where each node corresponds to a part of the document (such as elements, attributes, and text). The DOM allows programs (typically written in JavaScript) to interact with and manipulate the content, structure, and style of a webpage dynamically.

2.How do you select an element from the DOM?

To select an element from the DOM, methods like getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll() are used.

3.What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to handle events from its child elements. It improves performance and supports dynamically added elements by leveraging event bubbling.

4.How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using setAttribute(), getAttribute(), and removeAttribute(), while styles can be adjusted through the element's style property, modifying individual CSS properties.
